## Як запустити генерацію локально
1. Переконайтеся, що встановлено Python 3
2. Клонуйте репозиторій:
   ```
   git clone https://github.com/Kapusta5/-------.git
   ```
3. Запустіть скрипт:
   ```
   python lub2.py
   ```
4. Відкрийте згенерований `index.html` у браузері
